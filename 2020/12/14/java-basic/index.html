java是什么

java是一门面向对象编程语言，吸收了c++的优点,摈弃了c++的多继承、指针的复杂概念，java的拥有功能强大和简单易用的特性，其中Java中最大的特点是一切介对象，而且java的程序是运行在虚拟机上了，属于它具有跨平台的特性行，java的应用非常广，分布在软件很多领域，比如web,android,桌面应用等等
